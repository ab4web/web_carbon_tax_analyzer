# web_carbon_tax_analyzer
Stima elettricità, acqua, CO₂ e materiali critici consumati ogni anno da una pagina web
Pagina web statica composta da un unico file `index.html`.  
La pagina è pensata per essere leggera, veloce e facilmente integrabile in applicazioni o siti esterni.

## 🎯 Obiettivo del progetto
Fornire una pagina HTML minimale, ottimizzata per:
- caricamento rapido
- compatibilità con GitHub Pages
- integrazione in app o servizi web
- utilizzo come landing page, micro‑demo o contenuto informativo

## 📄 Contenuto del repository
- **index.html** – la pagina principale, completamente autonoma e senza dipendenze esterne.
